Current Web usage is perverted

  • Data usage is ambiguous

  • Same data scattered accross several applications / social networks / websites
  • Current attempts to fix this are not user-friendly
The Solid project is a specification

Data is stored in secure, decentralized storage units called Pods

Solid server & Pod
  • A solid server hosts one or multiple pods
  • Each pod is isolated
  • Each pod data and its access rules are unique
  • Fully controlled by pod owner (indivuals, companies...)
Authentication & permissions
  • Identity & permissions mechanisms
  • Permissions given on set & subset of data to other users & applications
  • Use OAuth & OpenID standards with some additions
  • Solid OIDC
