The Solid project is a specification üìú

Data is stored in secure, decentralized storage units called Pods

  • relies on standard protocols (TLS, RDF, HTML...)

Solid server & Pod

  • A solid server hosts one or multiple pods
  • Each pod is isolated
  • Each pod data and its access rules are unique
  • Fully controlled by pod owner (indivuals, companies...)
